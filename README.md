# Build Instructions
---
## Dependecies

* SFML 2
* OpenGL
* cmake
---
### Linux(ubuntu)

Install Dependencies

>sudo apt install libsfml-dev mesa-common-dev cmake

Change your working directory to CS202-Group-Project then

>mkdir build

>cd build

>cmake ..

>make

>cd ../bin

now run the program

>./platformer
